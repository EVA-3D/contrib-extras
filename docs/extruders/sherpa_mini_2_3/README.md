# Sherpa Mini Mount for EVA 2.3
You will need

* 2x M3x12 Socket Screw
* 2x M3x6 Socket Screw
* 4x M3x5 Heat Insert

.Step and .stl included. You have to print the main mount upside down with supports for the slot holes
